Module 1: Personal Website

Project Overview

The goal of this project is to build a personal developer website that showcases a biography, contact information, and details about Python projects. This project introduces the fundamentals of website development using Flask, a lightweight Python web framework.

Students will begin by learning to structure content with HTML, then enhance the presentation using CSS to style page elements. The website will serve as a creative portfolio, combining personal and professional information with project highlights.

Throughout the course, the site will be iteratively updated to reflect the student's expanding set of Python projects, resulting in a fully functional and personalized web presence by the end of the assignment.

Technologies Used

  • Python 3.10+
  • Flask Web Framework
  • HTML5 & CSS3
  • Jinja2 Templates
  • Flask Blueprints

Key Features

  • Responsive web design
  • Navigation bar with active tab highlighting
  • Professional biography and contact information
  • Project showcase with detailed descriptions
  • Clean, modern styling with CSS
